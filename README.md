This is designed to help students gain practice on Kinetics.  
There are two parts, the first is to determine reactant order based on the method of initial rates.  
They will then be asked to complete some follow up practice such as determining the value of the rate constant, k, with appropriate units, and determining the rate or concentration under different conditions.
The second part is using the integrated rate laws to determine the order of reaction based on graphical analysis.

The Link:  https://chemistrypractice.github.io/kineticsPractice/
